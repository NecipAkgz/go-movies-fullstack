## Getting Started

1. Start PostgreSQL using Docker Compose:

   ```shell
   docker-compose up -d
   ```

2. Navigate to the go-movies-backend directory:

   ```shell
   cd go-movies-backend
   ```

3. Run the application:
   ```shell
   go run ./cmd/api/
   ```
